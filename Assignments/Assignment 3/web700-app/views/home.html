Welcome to my website

I will be demonstrating HTML5 principles and techniques, as well as routing in Express. Explore the features of the website and how Express can handle different routes and views seamlessly.

About this Website

This site consists of 3 static views (Home, About, HTML Demo) as well as 5 dynamic routes that supply data. You can explore the following routes and views:

  • Home - The page you are currently on.
  • About - Learn more about this project.
  • HTML Demo - A demonstration of HTML5 elements.
  • All Students - View all students in the database.
  • Students by Course - View students in a Course (eg: course#03).
  • Student by Student Number - View details for Student (eg: StudentId#03).
  • All Managers - View all Teaching Assistants (TAs).
  • All Courses - View the list of all available courses.

HTML5 Overview

This site utilizes HTML5, the latest version of the HTML markup language. HTML5 is used for structuring and presenting content on the World Wide Web. It introduces new elements and APIs that simplify web development and ensure better compatibility across different browsers and devices.
